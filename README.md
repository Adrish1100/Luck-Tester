# 📖 Overview

The **Slot Machine Game** is a beginner-friendly web project developed using **HTML5, CSS3, and Vanilla JavaScript**. Inspired by classic casino slot machines, the game allows players to spin three reels while managing a virtual coin balance.

Each spin costs **50 coins**. Matching all three reels rewards the player with **100 coins**, while an unsuccessful spin deducts the betting amount. The game also includes reset functionality, spinning animations, and real-time status updates for an engaging user experience. 0 1

---

# ✨ Features

- 🎰 Animated three-reel slot machine
- 💰 Starting balance of **500 coins**
- 🎲 Random number generation for every spin
- 🏆 Win rewards when all reels match
- ❌ Lose coins on unsuccessful spins
- 🔄 Reset game at any time
- 📢 Live game status messages
- 🎨 Modern dark casino-themed interface

# 🎮 Game Rules

- Every player starts with **500 coins**.
- Press **SPIN** to play.
- Each spin costs **50 coins**.
- If all three reels display the same number:
  - ✅ You win **100 coins**.
- If the numbers do not match:
  - ❌ 50 coins are deducted.
- The **RESET** button restores:
  - Balance to **500 coins**
  - Reel values
  - Status message
- The game prevents spinning when the balance becomes too low.

# ▶️ Usage

1. Launch the game.
2. View your current balance.
3. Press **SPIN** to spin the reels.
4. Wait for the animation to finish.
5. Check the status message to see whether you won or lost.
6. Press **RESET** anytime to restart the game.
